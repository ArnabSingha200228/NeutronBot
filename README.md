# Neutron ☄️⭐🌟🌍👽🤖☄

Neutron is an AI-powered assistant built with **Streamlit** and **LangChain** that can chat with you, answer questions, and retrieve context from uploaded **files (.txt / .pdf)** or **web links**.  
It uses **Google Generative AI (Gemini)** for both chat and embeddings, and **ChromaDB** for vector storage.

---

## ✨ Features
- 💬 Chat with Neutron.
- 📂 Upload `.pdf` or `.txt` files → store embeddings → ask questions.
- 🌍 Paste website URLs → retrieve context → ask questions.
- 🔍 Retrieval-Augmented Generation (RAG) with ChromaDB.
- 🎨 Stylish UI with typing indicator & glowing header.

---

## ✨ Tools used
- 💬 Google Gemini
- 📂 langchain
- 🌍 pymupdf, urllib
- 🔍 ChromaDB
- 🎨 Streamlit, HTML, CSS
